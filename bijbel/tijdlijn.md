# Tijdlijn

## Wereld-chronologie

| Jaar | Gebeurtenis |
|------|-------------|
| 2024–2028 | Verdere politieke fragmentatie. AI neemt werk over in logistiek, administratie, transport. |
| 2028–2031 | Middenklasse-arbeid verdwijnt versneld. Sociale vangnetten onder druk. Klimaat: drie opeenvolgende droogtes in graangebieden. Voedselprijzen verdubbelen. |
| 2031 | Begin van The Contraction. Geen datum, geen aankondiging — later benoemd. |
| 2032 | Eerste grote pensioenfondscollapsen. Stadsmigratie versnelt. |
| 2033 | Politieke instabiliteit piek. Drie regeringswissels in zes maanden in vijf Europese landen. |
| 2034 | Elias Voss schrijft het ZVA-protocol (Zone Viability Assessment) voor The Accord Consortium. Bedoeld als diagnostisch triage-instrument. Wordt anders gebruikt. |
| 2034–2035 | The Accord-prototype ingezet in drie teststeden. Media: "De Stille Manager." |
| 2036 | The Accord officieel ingezet op continentale schaal. Elias Voss schrijft het allocatiemodel (eerste versie). |
| 2037–2038 | Elias ontdekt het compounding-mechanisme — zones gaan maar één kant op, geen correctievlak. Dient twee rapporten in. Formele bezwaar in comitévergadering: gedocumenteerd als "technical concern, noted." |
| 2038 | Elias Voss verlaat The Accord Consortium. Officiële reden: gezondheid. |
| 2039 | Einde van The Contraction — officieel. De wereld is smaller geworden. Genormaliseerd. |
| 2040 | Elias vestigt zich in Sector 9. Begint de tuin in het afgeschreven spoorwegterrein. Eerste bed: rode biet. |
| 2041 | Elias verdwijnt uit The Accord's locatiedata. Zijn dossier bestaat nog — zijn positie niet. Zijn formele adres in het systeem: een lege kamer op de vierde verdieping van een gedeeltelijk herbestemd blok. Niemand heeft gecontroleerd. |
| 2043 | Lena Voss (dochter van Elias, geboren 2003) trouwt. Hoge relevantiescore. Systeemintegratie-architect bij The Accord, Sector 2. Ze gelooft dat haar vader stierf in 2041 — er was een melding, een afsluiting in zijn dossier. Ze heeft nooit een lichaam gezien. Owen (de man die Elias zijn kamer in Fenwick Block verhuurde) overlijdt. |
| 2044 | Northern Expansion planning framework goedgekeurd door The Accord. Tienjarenplan voor logistieke en transitinfrastructuur in ondergebruikte zones. Sector 9 staat op de kaart als Priority Corridor — Phase 2 (start 2048). |
| 2045 | Mara begint als data-verificatrice bij The Accord, werkplek in Sector 9. |
| **2047** | **Verhaal begint.** |
