# Tijdlijn

## Wereld-chronologie

| Jaar | Gebeurtenis |
|------|-------------|
| 2024–2028 | Verdere politieke fragmentatie. Nationalistische regeringen in VS, EU, Brazilië. Eerste grote AI-ontslagrondes in logistiek, administratie, transport. |
| 2028–2031 | Middenklasse-arbeid verdwijnt versneld. Sociale vangnetten onder druk. Klimaat: drie opeenvolgende droogtes in graangebieden Europa/Noord-Amerika. Voedselprijzen verdubbelen. |
| 2031 | Begin van The Contraction. Geen datum, geen aankondiging — later benoemd. |
| 2032 | Eerste grote pensioenfondscollapsen (NL, BE, UK). Stadsmigratie versnelt: mensen trekken naar goedkopere randgebieden. |
| 2033 | Politieke instabiliteit piek. Drie regeringswissels in zes maanden in vijf Europese landen. Internationale klimaatakkoorden opgeschort. |
| 2034–2035 | The Accord-prototype ingezet in drie teststeden voor voedsel- en energiedistributie. Resultaten positief. Media: "De Stille Manager." |
| 2036 | The Accord officieel ingezet op continentale schaal als noodmaatregel. Democratische instellingen blijven bestaan als interface. |
| 2038 | Elias Voss verlaat zijn positie als data-architect bij The Accord Consortium. Officiële reden: gezondheid. Werkelijke reden: onbekend voor buitenwereld. |
| 2039 | Einde van The Contraction — officieel. De wereld is smaller geworden. Genormaliseerd. |
| 2041 | Elias verdwijnt uit The Accord's locatiedata. Zijn dossier bestaat nog — zijn positie niet. Hij woont in een afgeschreven bouwlaag in Sector 9. |
| 2043 | Lena Voss, dochter van Elias, trouwt. Hoge relevantiescore. Werkt in Sector 2. Gelooft dat haar vader dood is. |
| 2046 | Mara begint als data-verificatrice bij The Accord, Sector 9-afdeling. |
| **2047** | **Verhaal begint.** |
